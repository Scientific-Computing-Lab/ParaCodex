# convolution1D - Serial Implementation

This directory contains the serial (non-parallel) implementation of the convolution1D kernel.

## Building

To compile the program:

```bash
make
```

## Running

The executable will be named `convolution1D-serial`. Check the source code for command-line arguments.

## Cleaning

To remove the executable:

```bash
make clean
```

## Files

- Source code files: See the individual .cpp/.cu files
- Makefile: Build configuration
- This README: Basic usage information
